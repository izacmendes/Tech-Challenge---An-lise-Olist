# Tech Challenge - Análise Estratégica de Dados da Olist

## Sobre o Projeto

Este projeto tem como objetivo realizar uma análise estratégica dos dados da Olist, utilizando técnicas de análise de dados com Python e visualização de informações através do Power BI.

A análise busca identificar padrões relacionados ao crescimento do negócio, satisfação dos clientes, desempenho logístico e categorias de produtos com maior geração de receita.

---

## Tecnologias Utilizadas

* Python
* Pandas
* Power BI
* GitHub

---

## Bases de Dados Utilizadas

Foram utilizados os seguintes datasets disponibilizados pela Olist:

* Pedidos
* Itens dos Pedidos
* Avaliações
* Produtos
* Tradução de Categorias

---

## Principais Indicadores

* Pedidos Totais
* Receita Total
* Nota Média dos Clientes
* Percentual de Atraso

---

## Dashboards Desenvolvidos

### Visão Executiva

Análise geral do desempenho do negócio, incluindo evolução dos pedidos e da receita.

### Experiência do Cliente

Análise das avaliações dos consumidores e impacto dos atrasos logísticos na satisfação.

### Produtos e Categorias

Identificação das categorias responsáveis pela maior geração de receita.

---

## Principais Insights

* Crescimento consistente do volume de pedidos.
* Receita superior a R$ 13 milhões.
* Nota média dos clientes superior a 4 estrelas.
* Impacto significativo dos atrasos na satisfação dos consumidores.
* Forte concentração da receita em categorias relacionadas a Saúde e Beleza, Relógios e Presentes e Cama, Mesa e Banho.
